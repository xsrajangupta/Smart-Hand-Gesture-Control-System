# 🤖 Smart Hand Gesture Control System

A touchless computer control system that uses **Arduino Uno**, **Ultrasonic Sensors**, and **Python** to detect hand gestures and perform mouse or presentation controls.

---

## 📌 Features

- 👋 Touchless Hand Gesture Recognition
- 🖱️ Mouse Cursor Control
- ⏭️ Next Slide Gesture
- ⏮️ Previous Slide Gesture
- 📡 Real-time Ultrasonic Sensor Detection
- 💻 Python-Arduino Serial Communication
- 🎯 Low-cost and Easy to Build

---

## 🛠 Hardware Used

- Arduino Uno
- HC-SR04 Ultrasonic Sensors
- USB Cable
- Jumper Wires
- Breadboard

---

## 💻 Software Used

- Arduino IDE
- Python
- OpenCV
- PySerial
- PyAutoGUI

## 🚀 How It Works

1. Ultrasonic sensors detect hand movement.
2. Arduino calculates the hand position.
3. Sensor data is sent to Python via Serial Communication.
4. Python interprets gestures.
5. Mouse movements or presentation controls are executed.
